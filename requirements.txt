# EduSync Requirements & Installations

Since this is a Vite + React + Node.js application, we do not use a Python `requirements.txt`. Instead, dependencies are managed via `package.json` and `bun`.

## Prerequisites
1. Node.js (v18 or higher)
2. bun (usually comes with Node.js)

## Installation Commands

Run the following command in the terminal to install all required dependencies:
```bash
bun install
```

## Key Dependencies Used in this Project
* **React** (`react`, `react-dom`): UI Library.
* **React Router Dom** (`react-router-dom`): For navigating between HOD and Faculty portals.
* **Firebase** (`firebase`, `firebase-admin`): For Authentication (including OTP), Firestore (database), and Storage (PDFs).
* **Tailwind CSS** (`tailwindcss`, `autoprefixer`, `postcss`): For styling.
* **Shadcn UI & Radix UI** (`@radix-ui/*`, `lucide-react`): For pre-built accessible components like calendars, dialogs, and navigation menus.
* **React Hook Form & Zod** (`react-hook-form`, `@hookform/resolvers`, `zod`): For form validation (leave requests, sign-ups, etc.).
* **Date-fns** (`date-fns`): For date manipulation (required for leave calendar and event sorting).
* **React Day Picker** (`react-day-picker`): For the interactive leave application calendar.
* **TanStack React Query** (`@tanstack/react-query`): For caching and data fetching from Firebase.

## Setup Firebase
1. Create a Firebase project at console.firebase.google.com
2. Enable Authentication (Email/Password, Phone)
3. Enable Firestore Database
4. Enable Firebase Storage (for PDFs)
5. Copy your Firebase config to a `.env.local` file in the root directory:

```env
VITE_FIREBASE_API_KEY="your-api-key"
VITE_FIREBASE_AUTH_DOMAIN="your-auth-domain"
VITE_FIREBASE_PROJECT_ID="your-project-id"
VITE_FIREBASE_STORAGE_BUCKET="your-storage-bucket"
VITE_FIREBASE_MESSAGING_SENDER_ID="your-sender-id"
VITE_FIREBASE_APP_ID="your-app-id"
```

## Running the Project
```bash
bun run dev
```

This will start the local development server at `http://localhost:5173`.
